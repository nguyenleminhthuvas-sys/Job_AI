# QUY TRÌNH THẨM ĐỊNH TÍN DỤNG HOÀN CHỈNH DÀNH CHO DỰ ÁN VÀ DOANH NGHIỆP

## 1. Mục tiêu của Thẩm định Tín dụng

Xác định khả năng trả nợ từ dòng tiền dự án/doanh nghiệp bền vững, đánh giá toàn diện rủi ro tài chính, pháp lý, kỹ thuật và ESG. Quyết định dựa trên DSCR>1.25x stress-tested, LTV<70%, IRR>NWACC+5%. Áp dụng cho vay mới, gia hạn, vốn lưu động theo Basel III và Thông tư 39 NHNN [INFERENCE từ SOP gốc].

## 2. Sàng lọc Ban đầu \& Thu thập Thông tin (1-3 ngày)

### Tài liệu bắt buộc (Checklist)

- Hồ sơ pháp lý: Giấy ĐKDN/GPĐT, QHĐT, giấy phép môi trường
- Tài chính: Báo cáo kiểm toán 3-5 năm, DCF/NPVs, ngân sách dự án
- Đảm bảo: Giấy tờ TSĐB, báo cáo thẩm định độc lập
- Khác: Hợp đồng EPC, hợp đồng mua bán, CV quản lý chủ chốt


### Xác minh dữ liệu

- CIC, VCB-Trade, Cổng thông tin Đầu tư công
- Thực địa: Site visit, phỏng vấn khách hàng/nhà cung cấp
- **Red flags**: Chậm nộp thuế >30 ngày, nợ xấu CIC, thay đổi pháp lý


## 3. Thẩm định Toàn diện (5-10 ngày)

### 3.1 Hiểu Dự án \& Ngành

- **Mô hình kinh doanh**: Revenue model, CAPEX/OPEX breakdown
- **Ngành**: Porter 5 Forces, cyclicality index, ESG risks
- **Cạnh tranh**: Market share, pricing power, customer concentration <20%


### 3.2 Thẩm định Kỹ thuật \& Khả thi

- **NPV/IRR**: Base/best/worst case; sensitivity ±20% revenue
- **Lịch trình**: Critical path, milestones, penalty clauses
- **Nhà thầu**: Track record, financial strength, insurance


### 3.3 Thẩm định Pháp lý \& Đất đai

| Yếu tố | Checklist | Red flags |
| :-- | :-- | :-- |
| Giấy phép | GPĐT, QHĐT, PPLT đầy đủ? | Tranh chấp đất đai |
| Hợp đồng | PPA/EPC có ràng buộc? | Điều khoản bất lợi |
| Thuế | Xác nhận nộp thuế? | Phạt chậm nộp >100tr |

### 3.4 Phân tích Tài chính (SOP gốc + Dự án)

```
Revenue Quality: Organic vs Project-based
- Dự án: Take-or-pay ≥70%? Capacity utilization >80% Y3?
- Doanh nghiệp: AR turnover >industry avg, DSO<60 ngày
Cash Flow: FCF supports debt service + distributions
EBITDA normalized = Reported - One-offs + Sustainable addbacks
```

**Key Ratios cho Dự án**:


| Metric | Formula | Threshold | Implication |
| :-- | :-- | :-- | :-- |
| Project IRR | DCF Analysis | >12% | Phê duyệt ≥15% |
| DSCR | CFADS/Debt Service | >1.3x | Covenant 1.2x |
| LLCR | NPVs/Debt O/S | >1.5x | Reserve account |
| LTV | Loan/Asset Value | <70% | Step-up nếu >80% |

### 3.5 ESG \& Rủi ro Bền vững

- **E**: EIA approval, emission targets
- **S**: Local community impact, labor compliance
- **G**: Anti-bribery policy, board independence


## 4. Đánh giá Rủi ro \& Stress Test

```
Worst Case Matrix:
1. Revenue -25% + Margin -5pp → DSCR ?
2. Delay 6 months + Cost overrun 15% → LLCR ?
3. FX +20%/Interest +200bps → Coverage ?
4. Customer default top 3 → Reserve adequacy?
```

**Early Warning Triggers**: DSCR<1.1x, utilization<70%, covenant breach.

## 5. Cấu trúc Giao dịch \& Điều kiện

```
Loan Structure:
- Tenor: Match project cash flow curve (5-12y)
- Grace: Construction period +6m
- Repayment: Sculpted per CFADS
- Covenants: MAC clause, Min DSCR, Debt inc restrictions
- Security: 1st ranking project assets + Sponsor guarantee
```

**Conditions Precedent**:

- [ ] Financial close EPC/PPA
- [ ] Land clearance completed
- [ ] Insurance policies in place
- [ ] Equity injection verified


## 6. Quy trình Phê duyệt (Credit Committee)

```
Presentation Structure (15 slides):
1. Executive Summary (1pg rating recommendation)
2. Project Overview + Site photos
3. Financial Analysis + Charts
4. Stress Tests + Sensitivity
5. Risk Matrix + Mitigants
6. Structure + Covenants
7. Recommendation + Limits

Key Committee Questions:
• "Worst case DSCR trajectory?"
• "What kills the project?"
• "Exit strategy if sponsor default?"
• "Reserve account sizing?"
```


## 7. Giám sát Sau vay \& Early Warning

```
Quarterly Monitoring Pack:
- Actual vs Budget (variance >10% = flag)
- DSCR forecast 12m forward
- Covenant compliance certificate
- Material change notifications

Triggers → Actions:
DSCR 1.1-1.2x → Enhanced monitoring
DSCR <1.1x → Mandatory cure/remedy
```


## 8. Checklist Thẩm định Hoàn chỉnh (Credit Memo)

```
PHASE 1 - PRE-SCREENING ✓
□ Legal docs complete? CIC clean?
□ Site visit report? Initial ratios screen?

PHASE 2 - FULL ANALYSIS ✓  
□ DCF 3 scenarios? Stress tests passed?
□ TSĐB valuation <70% LTV?
□ EPC/PPA contracts reviewed?

PHASE 3 - COMMITTEE READY ✓
□ Risk mitigants quantified?
□ Alternative scenarios presented?
□ Management interviewed?

DECISION: [APPROVE/CONDITIONAL/REJECT]
Limits: [AMOUNT/TENOR/COVENANTS]
```
