# PDF to Markdown Parser

> Parse PDFs (books & financial reports) into LLM-ready Markdown

## What It Does

Specialized skill for extracting PDF content into **clean Markdown format** optimized for LLM consumption. Supports two modes:

- **Book Mode**: Ultra-fast extraction for books and long documents
- **Finance Mode**: Ultra-detailed extraction preserving tables, charts, and financial data

## Key Features

- 📚 **Book Mode**: High-speed parsing with chapter structure preservation
- 💹 **Finance Mode**: Detailed table extraction for financial statements
- 🔄 **LLM-Optimized Output**: Clean Markdown ready for AI processing
- 📊 **Table Preservation**: Accurate conversion of complex table layouts

`"bóc tách PDF"`, `"PDF to markdown"`, `"parse PDF"`, `"đọc PDF"`, `"chuyển PDF sang markdown"`
